# Lithium Exploration Potential in Québec

This repository presents a personal exploratory study on lithium exploration potential in Québec, Canada.

The project combines public mining datasets, spatial analysis and economic interpretation to identify areas of interest related to critical mineral resources.

## Project objective

The objective is to explore how public mining data can be used to identify and compare areas with potential interest for lithium exploration.

The study focuses on:

- lithium-related mineral occurrences;
- existing lithium projects;
- exploration targets;
- spatial proximity between indicators;
- economic and territorial interpretation of the results.

## Tools used

- Python
- QGIS
- Excel
- Public mining datasets from Québec

## Repository structure

The repository will include:

- `data/`: selected datasets and documentation of data sources;
- `scripts/`: Python scripts used for data processing and analysis;
- `outputs/`: maps, tables and results;
- `report/`: final report or synthetic document.

## Disclaimer

This project is an exploratory academic and personal study. It is not a geological feasibility study, nor an investment recommendation.

# Data sources

This folder documents the public datasets used in the project.

The study relies on public mining and geological datasets related to Québec, Canada.

## Main data categories

- Lithium-related mineral occurrences
- Existing lithium mining projects
- Exploration targets
- Spatial coordinates and territorial information

## Notes

Some raw datasets may not be fully uploaded to this repository if they are too large or if they are better accessed directly from official public data portals.
